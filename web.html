Grundlæggende Web

Opgaven:

I dette tema gennemgik vi de mange forskellige stilarter og blev introduceret til HTML og CSS. Opgaven lød på at udarbejde et responsivt site med HTML og CSS der skulle passe til vores tildelte stilart. Jeg fik tildelt stilarten, ”Bauhaus” og skulle derfor lave et grafisk udtryk på siden, som passet til stilarten Bauhaus.
